What are Pseudo-Elements?

CSS pseudo-element is used to style specified parts of an element. Lorem ipsum dolor sit amet consectetur adipisicing elit. Modi laudantium inventore tempore aperiam quidem omnis neque. Dolorum hic repellat est, et provident a, aspernatur deserunt accusantium, sed deleniti eveniet aut?

For example, it can be used to:

Style the first letter, or line, of an element

Insert content before, or after, the content of an element

There are 6 pseudo elements selectors, and are indicated using "::" - double colons

  • ::after
    • Used to insert something after the content
  • ::before
    • Used to insert something before the content
  • ::first-letter
    • Selects the first letter of the chosen element
  • ::first-line
    • Selects the first line of the chosen element
  • ::marker
    • Selects the markers of list items
  • ::selection
    • Selects the portion of an element that is selected by a user
